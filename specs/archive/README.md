# Specification Archive

This directory preserves completed and superseded design records. Archived
documents remain useful historical context, but active proposals and unresolved
design work belong in `specs/future-ideas.md` or a new RFC outside the archive.

## Implemented

- `implemented/async-iteration/rfc.md` — async generators, lifecycle-aware
  iteration, reductions, transduction, and manual source cleanup. Its notes and
  completed task checklist are stored beside it.
- `implemented/pre-1-0-promise-combinators-and-time-observation/rfc.md` —
  one-next-take Promise combinators and observation-only `time` semantics.
- `implemented/remove-implicit-try/rfc.md` — removal of implicit trailing
  `catch` / `finally` syntax.
- `completed-future-ideas.md` — completed decisions and cleanup work that began
  as entries in `specs/future-ideas.md` rather than focused RFCs.

## Superseded

- `superseded/promise-combinator-next-take/rfc.md` — incorporated into the
  implemented pre-1.0 Promise combinator and `time` RFC.
